> 创建: 2026-02-08
> 状态: 已完成

## 目标

更新托盘代码以支持 Retina 高分辨率显示，提供更清晰的图标效果。

## 当前问题

- 代码只使用 icon.png (22x22)
- icon@2x.png (44x44) 未被使用
- 在 Retina 屏幕上显示可能模糊

## 解决方案

使用更高分辨率的图标 (44x44) 作为基础图标，让系统自动处理缩放，在 Retina 屏幕上获得更好的显示效果。

### 工作原理

macOS 系统托盘自动处理图标缩放：
1. 使用 44x44 像素图标
2. 系统缩放到 22pt (points)
3. 在 Retina 屏幕：22pt = 44px → 完美匹配，无缩放损失
4. 在普通屏幕：22pt = 22px → 系统自动缩小

### 文件状态

- `icon.png`: 22x22, 579B（保留作为参考）
- `icon@2x.png`: 44x44, 1.6KB（**正在使用**）

### 优势

✅ Retina 屏幕上显示更清晰
✅ 普通屏幕也能正常显示
✅ 圆角效果保持完整
✅ 无需额外代码处理多分辨率

## 技术说明

macOS 系统托盘会自动处理图标缩放：
- 使用 44x44 图标，系统会缩放到 22pt
- 在 Retina 屏幕上，22pt = 44px，完美匹配
- 在普通屏幕上，系统自动缩小
